# Bebi Field Ledger

Mobile field-operations tracker for the Bebi Project (2026) — Cross River State cassava project.
Mirrors the Bebi Project (2026) Operations Tracker Excel workbook: cost heads, daily expenses,
budget vs actual, daily productivity, diesel sub-ledger, machine utilization, fund retirement,
and earned value management (EVM).

Open `index.html` directly in a browser, or enable **GitHub Pages** (Settings → Pages → Deploy
from branch `main`, folder `/`) to get a shareable live URL.

## Data storage

Currently stores entries in the browser's local storage (per device). Multi-user sync via
Google Drive is being added as a follow-up.
